Este proyecto es una sistema de gestión de tickets de IT. Incluye una **API REST** para las peticiones, haciendo uso de un webhook con **[make.com](make.com)**

## Tecnologías

- Node.js + Express
- TypeScript
- Docker
- Google Sheets API
- Make.com (webhooks)
- GitHub Copilot
